====================
으로 [Chris Pels](https://twitter.com/chrispels)

이 비디오에서는 Chris Pels에 데이터베이스 또는 다른 원본에서 데이터를 가져오는 및 Excel 같은 응용 프로그램에서 사용할 수 있는 쉼표로 구분 된 파일 내보내기 방법을 보여 줍니다. 먼저, DataTable 개체와 데이터 집합이 생성 됩니다. 다음으로, 현재 웹 페이지 요청에 대 한 응답의 선택을 취소 하 고 헤더 및 콘텐츠 형식 csv 파일로 구성 됩니다. 그런 다음 실제 데이터는 첫 번째 데이터 값과 csv 파일에 대 한 열 헤더를 작성 하 여 응답 스트림에 추가 됩니다. 이 방법은 사용자가 로컬로 Excel 같은 프로그램에서 조작할 수 있도록 데이터의 내보내기를 필요로 하는 경우 유용할 수 있습니다.
