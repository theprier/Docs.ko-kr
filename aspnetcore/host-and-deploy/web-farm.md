* <span data-ttu-id="555cf-113">**유지 관리** &ndash; 웹 팜의 노드는 공유 서비스 집합을 사용하므로 시스템을 더 쉽게 관리할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="555cf-113">**Maintainability** &ndash; Nodes of a web farm can rely on a set of shared services, which results in easier system management.</span></span> <span data-ttu-id="555cf-114">예를 들어 웹 팜의 노드는 단일 데이터베이스 서버 및 정적 리소스(예: 이미지 및 다운로드 가능한 파일)의 일반 네트워크 위치를 사용할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="555cf-114">For example, the nodes of a web farm can rely upon a single database server and a common network location for static resources, such as images and downloadable files.</span></span>
